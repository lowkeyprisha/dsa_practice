# DSA Masterplan

A structured and interactive DSA roadmap built for coding interview preparation. This project organizes problems pattern-wise to improve problem-solving skills, consistency, and interview readiness.

---

# 🚀 Features

* 📚 Pattern-based DSA learning
* 🔍 Categorized problems for focused practice
* 📈 Scalable progress tracking architecture
* ⚡ Fast and responsive React UI
* 🎯 Interview-focused roadmap
* 🛠 Easily customizable and extendable

---

# 🧑‍💻 Skills & Technologies Used

## Frontend Skills

* React.js
* JavaScript (ES6+)
* JSX
* Component-based architecture
* Dynamic rendering
* Responsive UI design

## DSA Skills Covered

* Arrays & Hashing
* Sliding Window
* Binary Search
* Linked Lists
* Trees & Graphs
* Dynamic Programming
* Greedy Algorithms
* Backtracking

# ⚙️ Getting Started

## Clone the Repository

```bash
git clone https://github.com/lowkeyprisha/dsa_practice.git
cd dsa_practice
```

## Install Dependencies

```bash
npm install
```

## Run the Project

```bash
npm run dev
```
